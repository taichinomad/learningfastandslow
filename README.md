# Learning Fast and Slow

A public laboratory for learning.

This repository collects experiments, notes, and small projects exploring how learning actually happens. Some things here are quick explorations. Others are slow attempts at understanding.

The idea is simple:

- fast learning for exploration
- slow learning for depth

Both are necessary.

## What you may find here

- notes from books and ideas
- small software experiments
- learning frameworks and reflections
- prototypes and tools
- half-finished thoughts

Some things will be rough. That is intentional. This is a place for thinking in progress.

## Why this exists

Most learning on the internet is optimized for speed: summaries, hacks, shortcuts.

But deep understanding takes time.

This project explores the balance between the two.

## Principles

- learn in public
- experiment often
- think slowly
- keep things simple
- share what works
